# 🏙️ Smart City Management System — Islamabad

A C++ console-based Smart City Management System developed as a Data Structures course project.
Built using 7 custom data structures loaded with real Islamabad data.

## 🗂️ Data Structures Built From Scratch

| Data Structure | File | Used For |
|---|---|---|
| Graph (Adjacency List) | Graph.h | Bus routes and transport network |
| Min Heap / Priority Queue | MinHeap.h | Hospital priority ranking |
| Hash Table | IntHashTable.h | Fast lookup for citizens, schools, malls |
| Linked List | LinkedList.h | Dynamic data storage |
| Queue | Queue.h | Patient and service queues |
| Stack | Stack.h | Navigation operations |
| N-ary Tree | Tree.h | Islamabad hierarchy, schools, population |

## 🚀 Systems Included
- 🚌 Transport — bus routes, stops, shortest path
- 🏫 Education — school ranking and search
- 🏥 Medical — hospital priority queue, pharmacy lookup
- 👥 Population — citizen data management
- 🏬 Commercial — mall and facility search

## 🛠️ Requirements
- Windows OS
- Visual Studio 2019 or later
- No external libraries — everything built from scratch!

## ▶️ How to Run
1. Open Visual Studio
2. Open folder with all `.h` files and `Source.cpp`
3. Make sure `data/` folder with CSV files is in the same directory
4. Build → Ctrl + Shift + B
5. Run → F5
